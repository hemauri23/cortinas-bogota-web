# Cortinas Bogotá – Sitio Web Corporativo

Proyecto web desarrollado para una empresa de cortinas y persianas a la medida en Bogotá.

## Descripción
Sitio web informativo y comercial enfocado en la captación de clientes mediante formularios,
botones de contacto por WhatsApp y presentación de productos y servicios.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap
- PHP (formularios)
- Diseño Responsive

## Funcionalidades principales
- Página corporativa
- Formulario de contacto
- Botón de WhatsApp dinámico
- Secciones de productos y servicios
- Diseño adaptable a móviles y escritorio

## Rol desempeñado
- Análisis de requerimientos
- Desarrollo frontend
- Integración de formularios
- Optimización de experiencia de usuario
